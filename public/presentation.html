Apple 3D UI Previewer

Preview UI mockups in the browser with real-time 3D, orbit to any viewing angle, and export PNG. Built for workflows that need multi-angle, archivable presentation assets—instead of a single flat comp, fixed-angle device frames, or redoing Photoshop composites every time.

Overview & rationale

In proposals, reviews, and print, UI work is often shown with 2D comps or fixed-angle device mockups. Changing the angle usually means re-mapping textures, perspective, and light. Many tools stop at a single “hero still,” making it hard to judge whether hierarchy and brand elements still read under the same scale and lighting when the camera moves.

This tool frames the problem as: preview the interface in a controlled 3D scene and close the loop with PNG export—so multi-angle assets are the default, not a one-off composite every time.

Design note

Unlike “live in the scene editor” tools, interactions prioritise set angle → export → reuse; the stack is chosen for stable in-browser execution and static hosting.

Role in visual communication workflows

  • Proposals & pitches — Export PNGs from multiple viewing angles for decks, PDFs, or print.
  • Review & iteration — Orbit to check hierarchy and whitespace in 3D composition, not only a single mockup still.
  • Asset handoff — Versionable stills; new angles do not require redoing the entire composite pipeline.
  • Pipeline fit — Screen design stays in 2D tools; this covers flat artwork → multi-angle device presentation.

Design thinking

Problem framing. Many workflows force a choice between flat output and fixed-angle templates. What is missing is a lightweight path for the same content, continuously adjustable angle, PNG as the deliverable.

Direction. In-browser 3D preview plus an interaction model oriented to export, reducing reliance on heavy DCC apps or per-frame Photoshop work.

UI/UX decisions

The following choices map directly to multi-angle PNG delivery and legibility.

Decision Rationale
Orbit + PNG export Core loop is find the angle → export; output is tied to multi-angle value.
Grouped screen-mapped adjustments Brightness, scale, nudge, etc. directly affect each export.
Global / chromeless preview Maximise frame control before export; keep UI chrome out of shot.
Fit / Fill & orientation Surfaces crop and aspect explicitly for discussion under different media constraints.
English / 中文 UI Lowers friction for mixed-language reviews and handoffs.
